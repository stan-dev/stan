Detailed Description

template<class M, class Q, class BaseRNG>
class stan::variational::advi< M, Q, BaseRNG >

AUTOMATIC DIFFERENTIATION VARIATIONAL INFERENCE.

Runs "black box" variational inference by applying stochastic gradient ascent in order to maximize the Evidence Lower Bound for a given model and variational family.

Template Parameters
Mclass of model
Qclass of variational distribution
BaseRNGclass of random number generator
Parameters
mstan model
cont_paramsinitialization of continuous parameters
n_monte_carlo_gradnumber of samples for gradient computation
n_monte_carlo_elbonumber of samples for ELBO computation
eta_adagradeta parameter for adaGrad
rngrandom number generator
eval_elboevaluate ELBO at every "eval_elbo" iters
n_posterior_samplesnumber of samples to draw from posterior
print_streamstream for convergence assessment output
output_streamstream for parameters output
diagnostic_streamstream for ELBO output
