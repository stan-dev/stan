Stan: Bugs & Features

Reporting Bugs

To report bugs, please use

  • the Stan Issue Tracker.

To make it as easy as possible for us to help you, please provide as much context as possible, including the version of RStan and R, C++ compiler version, and operating system (including bit size and memory).

Requesting Features

To formally request a new feature for Stan, please also use

  • the Stan Issue Tracker.

We informally discuss feature requests on both

  • the user's and developer's mailing lists.
